# Royal Engagement Invitation Website

This is a static landing page ready for deployment on **GitHub Pages** or **Netlify**.

## Files
- `index.html` – main website file
- `assets/invitation.jpg` – invitation image
- `assets/music.wav` – original ambient background music loop

## Quick Deploy
### GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In GitHub repo settings, enable **Pages** and select the main branch.

### Netlify
1. Drag and drop this folder into Netlify Deploys, or connect the repository.
2. No build command needed.
3. Publish directory: `.`

## Optional Edits
- Replace `assets/invitation.jpg` with additional images if you want.
- If you want live form submissions on Netlify, keep the existing form markup in `index.html`.
